## Overview

This example use case demonstrates how to automate team recognition by creating daily PR celebration workflows. You'll learn how to:

- Fetch merged pull requests from GitHub
- Analyse all the requests
- Post updates to Slack channels
- Schedule it

## Step 1 - Enter the PR Request Prompt

Enter the prompt: "get me the new PRs merged approved yesterday in the main branch in runableHQ repo." 

![PR Request Prompt](https://usercontent.in.prod.clueso.io/138e6e12-51a4-4458-9914-1bc6359a8384/c3cb82fd-0365-4800-9910-5cac56195fbc/5639abb2-df44-4f97-9ce0-8126ccdc3dec/images/36e3d3fe-f104-4c0a-99bb-a5555d9abfe0.gif)


## Step 2 - Verify GitHub Connection and Collect Merged PRs

Runable checks for a connection to your GitHub account. Ensure your GitHub is linked so PR data can be pulled accurately.

The system collects yesterday's merged PRs, with reviewer details, and saves them in a markdown file for tracking. 

![PR Data Saved](https://usercontent.in.prod.clueso.io/138e6e12-51a4-4458-9914-1bc6359a8384/c3cb82fd-0365-4800-9910-5cac56195fbc/5639abb2-df44-4f97-9ce0-8126ccdc3dec/images/5b18051c-aebd-4f08-b11c-c3e9fbe7333e.png)


## Step 3 - Compose and Send Congratulatory Message to Slack

prompt: "write a small congrats message in the 'all-runable' slack channel mentioning the authors in bold and PR links."

Runable connects to your Slack workspace and sends your crafted message to the designated channel, recognizing contributors and linking PRs. 

![Slack Connection](https://usercontent.in.prod.clueso.io/138e6e12-51a4-4458-9914-1bc6359a8384/c3cb82fd-0365-4800-9910-5cac56195fbc/5639abb2-df44-4f97-9ce0-8126ccdc3dec/images/f19426b9-dc32-4e16-8538-dcd3fa083d0d.gif)


## Step 4 - Verify Message in Slack

Check the Slack channel to verify your message has posted correctly, so everyone's achievements are visible. 

![Message in Slack](https://usercontent.in.prod.clueso.io/138e6e12-51a4-4458-9914-1bc6359a8384/c3cb82fd-0365-4800-9910-5cac56195fbc/5639abb2-df44-4f97-9ce0-8126ccdc3dec/images/12d27caa-dacc-48e7-8d7b-256960a7dcbf.gif)
