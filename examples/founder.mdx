## Overview

This example use case demonstrates how to automate team recognition by creating daily PR celebration workflows. You'll learn how to:

- Fetch merged pull requests from GitHub
- Generate congratulatory messages
- Post updates to Slack channels
- Automate the entire recognition process
- Schedule daily team celebrations

Open Runable to begin automating your daily pull request workflow. The animated GIF below visually demonstrates the exact steps to get started—watch the animation to see how to access and prepare the workflow.
