## Overview

This example use case demonstrates how to automate stock research and financial analysis using Runable. You'll learn how to:

- Gather comprehensive financial data
- Analyze stock performance metrics
- Document research findings
- Export to Google Docs
- Share insights via email

## Step 1 - Start with a Detailed Research Prompt

Prompt that asks for essential stock data points like EPS, Revenue, Free Cash Flow for the last four quarters, Valuation analysis, and Risk Assessment. 

![Research Prompt](https://usercontent.in.prod.clueso.io/138e6e12-51a4-4458-9914-1bc6359a8384/1336845d-cefd-4a38-be04-273abb7af8db/4e7f0057-5279-4950-bd56-5d0bfbd23485/images/5f9065d7-3ce6-4c93-b6db-3d69bcb7be98.gif)


## Step 2 - Let the Agent Perform Deep Research and Save File

Now Runable pulls in data according to your prompt.

![Research Process](https://usercontent.in.prod.clueso.io/138e6e12-51a4-4458-9914-1bc6359a8384/1336845d-cefd-4a38-be04-273abb7af8db/4e7f0057-5279-4950-bd56-5d0bfbd23485/images/737e60f8-03d5-40dd-b68e-e94e2a65a623.gif)

And save the findings in a markdown file, making your results easy to reference or reuse.
</Tip>

## Step 3 - Export Research to Google Docs

First the agent will see if we are conencted to google docs if not then ask user to do that and then type "continue" and click enter. 

![Google Docs Connection](https://usercontent.in.prod.clueso.io/138e6e12-51a4-4458-9914-1bc6359a8384/1336845d-cefd-4a38-be04-273abb7af8db/4e7f0057-5279-4950-bd56-5d0bfbd23485/images/c46f143f-63a1-4fb7-aa68-d0fa1b538711.png)

Now the agent is connected to your google docs. It can upload the date to it. 

![Upload Process](https://usercontent.in.prod.clueso.io/138e6e12-51a4-4458-9914-1bc6359a8384/1336845d-cefd-4a38-be04-273abb7af8db/4e7f0057-5279-4950-bd56-5d0bfbd23485/images/082d52c3-500f-4c3e-ab1e-03b10356504d.gif)

## Step 4 - Verify Your Research in Google Docs

Open your Google Docs to confirm the research document is there and contains all the expected insights.

![Google Docs Verification](https://usercontent.in.prod.clueso.io/138e6e12-51a4-4458-9914-1bc6359a8384/1336845d-cefd-4a38-be04-273abb7af8db/4e7f0057-5279-4950-bd56-5d0bfbd23485/images/1c3688f1-41a6-4c43-92a8-28ade0087c50.png)

## Step 5 - Email Research with Important Details and Link

Direct the agent to send an email containing your research summary and the Google Doc link.
